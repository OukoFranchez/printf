# 0x11. C - printf 

## Authors

	. Ouko Ouko
	. Eric Ubaga

## Requirements

	. Editors: vi, vim, emacs
	. Ubuntu 20.04 LTS and gcc, using the options -Wall -Werror -Wextra -pedantic -std=gnu89
	. Betty style compliant code. It will be checked using betty-style.pl and betty-doc.pl
	. No global variables
	. No more than 5 functions per file
	. The only C standard library functions allowed are malloc, free and exit. Any use of functions like printf, puts, calloc, realloc etcâ€¦ is forbidden.
	. You are allowed to use `_putchar`
	. The prototypes of all your functions should be included in header file `main.h`
	. All your header files should be include guarded

## Authorized functions and macros

	. write (man 2 write)
	. malloc (man 3 malloc)
	. free (man 3 free)
	. va_start (man 3 va_start)
	. va_end (man 3 va_end)
	. va_copy (man 3 va_copy)
	. va_arg (man 3 va_arg)

## Compilation

	. Your code will be compiled this way:

		`$ gcc -Wall -Werror -Wextra -pedantic -std=gnu89 *.c`
